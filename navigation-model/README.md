# Spatial Audio Navigator

AI-powered navigation assistant for visually impaired individuals, specifically designed for a hackathon demo.

## Features

- **Camera Detection:** Real-time object and obstacle detection using TensorFlow.js and COCO-SSD.
- **Spatial Audio:** Directional audio guidance using Howler.js.
- **Voice Commands:** Hands-free control with Web Speech API.
- **Haptic Alerts:** Vibration feedback for nearby obstacles using Navigator Vibration API.
- **Futuristic UI:** Stunning dark theme, neon colors, glassmorphism, and Three.js radar animation.

## Tech Stack

- **Frontend:** React, Vite, React Router DOM, Tailwind CSS (v4)
- **AI/Vision:** TensorFlow.js, COCO-SSD
- **Audio/Speech:** Web Audio API, Howler.js, Web Speech API
- **Animations/3D:** Framer Motion, Three.js, React Three Fiber, React Three Drei
